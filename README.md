# Premium Local Web Host
A clean, modern PyQt-based GUI tool that lets you host any folder or web file (HTML, JS, CSS, etc.) on a local or LAN-accessible HTTP server with one click.

Perfect for:
- Web developers previewing projects
- Testing on mobile devices
- Quick LAN sharing
- Local demos and prototypes

---

## ✨ Features

### ✔ Choose a file or entire folder  
Serve a single HTML file or an entire project directory.

### ✔ Localhost or LAN mode  
- **Local Host** → Only accessible on your machine  
- **Global Network Host** → Accessible from phones, tablets, and other PCs on your WiFi/LAN

### ✔ Auto-open browser  
Automatically launches your default browser when the server starts.

### ✔ Clean, modern UI  
Dark theme, premium layout, and a proper SVG gear icon for settings.

### ✔ Clear selection  
Reset your chosen file/folder instantly.
