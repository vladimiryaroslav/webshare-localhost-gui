Installation

Get up and running in just a couple of commands. All you need is Python and PyQt5.

①
Install Python 3.8+

Download and install Python 3.8 or newer from the official website:
python.org

1 # Download the latest Python 3.8+ installer from python.org
②
Install dependencies

Install PyQt5 using pip:

1 pip install PyQt5

Once installed, launch Premium Local Web Host and start serving your projects instantly.
